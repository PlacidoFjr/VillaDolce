## Objetivo

Criar um site institucional completo, elegante e responsivo para a **Villa Dolce Ateliê**, um ateliê de cestas afetivas e delícias artesanais.

O site deve transmitir:

- Luxo minimalista.
- Carinho.
- Cuidado artesanal.
- Sofisticação.
- Memória afetiva.

A ação principal do cliente deve ser **pedir orçamento pelo WhatsApp**.

## Tecnologia

Use **HTML, CSS e JavaScript simples**, sem frameworks.

Organize os arquivos de forma clara e fácil de editar.

Estrutura esperada:

```text
index.html
catalogo.html
sobre.html
contato.html
css/style.css
js/script.js
assets/
```

## Identidade Visual

Baseie todo o visual na logo.

Use:

- Fundo creme/off-white.
- Marrom chocolate.
- Bege elegante.
- Dourado suave.
- Preto ou marrom escuro para textos.
- Muito espaço em branco.
- Linhas finas.
- Divisórias delicadas.
- Detalhes discretos inspirados em papelaria premium.

O site deve parecer uma **boutique artesanal refinada**, não uma loja comum.

Evite:

- Visual infantil.
- Excesso de cor.
- Muitos corações.
- Elementos genéricos.
- Aparência amadora.

## Tipografia

Use uma fonte serifada elegante para títulos e uma fonte limpa para textos.

O nome **Dolce** pode ter destaque visual com estilo mais caligráfico/script, desde que continue legível.

## Páginas

### 1. Home

Criar uma página inicial sofisticada com:

- Hero elegante.
- Destaque para a logo e nome **Villa Dolce Ateliê**.
- Frase principal: **“Cestas afetivas e delícias artesanais feitas para transformar gestos em memórias.”**
- Botão principal **“Pedir orçamento pelo WhatsApp”**.
- Prévia das categorias.
- Seção curta sobre a essência da marca.
- Diferenciais.
- CTA final para encomendas personalizadas.

### 2. Catálogo

Criar catálogo sem preços, com categorias premium:

- Cestas Afetivas.
- Caixas Presenteáveis.
- Delícias Artesanais.
- Datas Especiais.
- Personalizados.

Cada categoria deve ter:

- Nome.
- Descrição curta e sofisticada.
- Ocasiões indicadas.
- Placeholder para imagem real.
- Botão **“Consultar pelo WhatsApp”**.

Deixar claro que cada criação é personalizada e que o orçamento é feito individualmente.

### 3. Sobre

Criar página institucional contando a história da marca com base nesta ideia:

> A Villa Dolce nasceu do desejo de transformar pequenos gestos em lembranças inesquecíveis. Cada criação é pensada com carinho, cuidado e significado, para que quem recebe sinta afeto em cada detalhe.

A página deve transmitir:

- Afeto.
- Sofisticação.
- Cuidado artesanal.
- Confiança.
- Atenção aos detalhes.

Incluir valores:

- Delicadeza.
- Personalização.
- Cuidado.
- Afeto.
- Memórias especiais.

### 4. Contato

Criar página de contato com:

- Chamada para encomendas personalizadas.
- Botão de WhatsApp.
- Instagram: `@villadolceatelie`.
- Campos informativos editáveis para cidade, horário de atendimento e observações sobre pedidos.
- Texto curto convidando o cliente a falar sobre ocasião, estilo do presente e preferências.

## WhatsApp

Use o placeholder `[INSERIR_WHATSAPP]` em todos os links.

A mensagem pré-pronta deve ser:

```text
Olá, vim pelo site da Villa Dolce Ateliê e gostaria de informações sobre uma encomenda personalizada.
```

Criar os links no formato correto para WhatsApp, usando `encodeURIComponent` no JavaScript ou URL codificada no HTML.

## Interações

Adicionar animações elegantes e marcantes, mas sem exagero:

- Entrada suave das seções.
- Hover premium em botões e cards.
- Transições delicadas.
- Menu mobile funcional.
- Rolagem suave quando fizer sentido.

## Responsividade

O site deve funcionar muito bem no celular, tablet e desktop.

Apesar de ser equilibrado para desktop e mobile, garantir que a experiência no celular seja excelente, pois muitos clientes virão do Instagram.

## Imagens

Copie a logo para a pasta `assets` e use no site.

## Qualidade Esperada

- Código limpo e organizado.
- Visual premium e coerente com a logo.
- Textos em português do Brasil, com tom sofisticado e afetivo.
- Navegação clara entre Home, Catálogo, Sobre e Contato.

- Estrutura preparada para crescer futuramente com novos produtos, novas categorias ou loja online.
